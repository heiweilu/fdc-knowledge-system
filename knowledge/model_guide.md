# Simulink模型结构解读 — MMCSIDUAN0x11.slx

## 模型总览

**模型名称**：MMCSIDUAN0x11（MMC四端/两端系统）
**仿真工具**：MATLAB/Simulink
**求解器设置**：离散（Discrete），步长0.0001s（100μs）
**仿真时间**：20s
**停止时间**：20s

### 顶层结构

模型顶层包含两个核心子系统，通过直流线路互联：

```
┌──────────────────┐     直流正极线     ┌──────────────────┐
│                  │ ═══════════════════ │                  │
│  MMC1定直流电压    │                    │  MMC2定有功功率1   │
│  （A站/主控站）    │                    │  （B站/从控站）    │
│                  │ ═══════════════════ │                  │
└──────────────────┘     直流负极线     └──────────────────┘
```

- **MMC1定直流电压**：主控站，负责维持直流母线电压恒定
- **MMC2定有功功率1**：从控站，控制有功功率的传输

## MMC1定直流电压子系统详解

### 内部模块构成

从截图分析，MMC1内部包含以下关键模块：

**1. 交流侧电源与变压器**
- 三相交流电源（左下角）
- 升压/降压变压器（接线组别区分）
- 三相VI测量模块（Three-Phase V-I Measurement）

**2. MMC本体（中央灰色大块）**
- 多端口子系统，包含6个桥臂（三相上下桥臂）
- 输入信号：6组PWM开关信号（PWMAP, PWMAL, PWMBP, PWMBL, PWMCP, PWMCL）
- 输出信号：
  - 三相交流电压电流（IAP, IAL, IBP, IBL, ICP, ICL）
  - 三相阀侧电压（VAP, VBP, VCP）
  - 子模块电容电压（VCU, VBU, VAL, VDC等）
  - 直流侧电压电流（IDC, VDC, VDCS）
- 100:10变比（可能是100kV:10kV或对应标幺值）

**3. NLM调制模块**
- 最近电平逼近调制（NLM）子系统
- 输入：参考电压波形（VA, VB, VC来自控制环路）+ 子模块电容电压
- 输入信号包括：[VAL], [VBL], [VCL], [HLA], [HLB], [HLC] 等
- 输出：6组PWM脉冲信号给MMC本体
- 内含排序均压算法

**4. 控制系统（右下区域）**
- **PLL锁相环**：从交流侧电压获取同步角度
- **外环控制**：
  - 直流电压环（定Udc）：子模块电容电压→直流电压调节
  - 目标：[VAP] 端口输出稳定直流电压
- **内环电流控制**：
  - d轴电流控制（有功）
  - q轴电流控制（无功）
  - 输出参考电压给NLM
- **坐标变换**：abc/dq变换模块

**5. 测量与显示（右侧）**
- Scope观测器（V1, V2端子）
- 直流侧电压电流测量
- 交流侧P/Q功率计算（Vabc→P, Iabc→Q → Scope19）
- 信号输出到工作空间

### 关键信号标注

| 信号名 | 含义 | 位置 |
|--------|------|------|
| [AP], [AL] | A相上/下桥臂电流 | MMC输出→右侧 |
| [BP], [BL] | B相上/下桥臂电流 | MMC输出→右侧 |
| [CP], [CL] | C相上/下桥臂电流 | MMC输出→右侧 |
| [VAP], [VBP], [VCP] | 阀侧三相电压 | MMC输出 |
| [VCU], [VBU] | 子模块电容电压 | MMC输出→NLM |
| [VAL], [VDC] | 交流阀侧电压/直流电压 | 测量→控制 |
| [IDC] | 直流电流 | 测量输出 |
| PWMAP~PWMCL | 6组PWM脉冲 | NLM→MMC |
| HLA, HLB, HLC | 桥臂参考电压 | 控制→NLM |
| dref, qref | dq轴电流参考值 | 外环→内环 |

## MMC2定有功功率子系统详解

### 结构对比

MMC2的内部结构与MMC1基本一致，包含相同的模块：
- MMC本体（三相六桥臂）
- NLM调制模块
- 三相交流电源 + 变压器
- 控制系统

### 关键差异

**外环控制不同**：
- MMC1外环：**定直流电压**（Udc_ref → PI → id_ref）
- MMC2外环：**定有功功率**（P_ref → PI → id_ref）

MMC2中可以看到：
- 功率参考值信号（Mds0, Mqs0 等）用于设定有功功率目标
- 外环接收有功功率测量值（wt），通过PI调节器产生d轴电流参考
- 区别于MMC1的直流电压反馈环路

### 测量模块

与MMC1类似：
- Vabc2, Iabc2 → P/Q功率计算 → Scope19
- 直流侧电压、电流测量输出

## 仿真结果解读

### MMC1侧P/Q波形特征（MMC1定直流电压站）

从仿真结果截图分析（图：P,Q波形，y轴×10⁹，x轴0-20s）：

**有功功率wt（黑色曲线）：**
- 稳态值约 0.3×10⁹ W = 300MW
- 暂态过程：0-2s内有较大振荡（初始过冲至4×10⁹）
- 2s后逐渐收敛，带有持续的纹波

**无功功率Q（橙色曲线）：**
- 稳态值约 1.0×10⁹ Var = 1000MVar（偏大）
- 暂态过程中有负向过冲至-1.5×10⁹
- 纹波较大，包络宽约0.3×10⁹

**诊断要点**：
- 暂态时间约2-4s，对于MMC系统属于偏长
- 无功功率稳态值偏大（1GVar），需检查无功参考值设定
- 持续纹波可能与子模块数不足或电容值偏小有关

### MMC2侧P/Q波形特征（MMC2定有功功率站）

从仿真结果截图分析（图：P,Q波形，y轴×10⁸，x轴0-20s）：

**有功功率wt（黑色曲线）：**
- 稳态值约 -0.5×10⁸ W = -50MW（负值表示从直流侧吸收功率传到交流侧）
- 暂态过程：0-4s有明显振荡，最大负向过冲至-8×10⁸
- 4s后基本稳定

**无功功率Q（橙色曲线）：**
- 稳态值接近0（符合定Q=0的设定）
- 暂态过程中有正向过冲

**诊断要点**：
- MMC2的功率量级（~50MW）与MMC1（~300MW）差异较大，需确认设计容量
- 暂态振荡收敛时间约4s，可通过调节PI参数改善
- 无功基本为零，说明Q环控制正常

## 如何在MATLAB中查看详细参数

1. 打开MMCSIDUAN0x11.slx模型
2. 双击"MMC1定直流电压"子系统进入
3. 双击MMC本体（灰色大块）→查看子模块数、电容值
4. 查看桥臂电感：双击电感模块读取L值
5. 查看控制器参数：双击PI模块读取Kp、Ki
6. 查看NLM模块：双击进入查看调制算法参数
7. 查看交流源参数：双击三相源查看电压幅值和频率
