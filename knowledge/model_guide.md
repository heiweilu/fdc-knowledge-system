# Simulink模型结构解读 — MMCSIDUAN0x11.slx

## 模型总览

**模型名称**：MMCSIDUAN0x11（MMC四端/两端系统）
**仿真工具**：MATLAB/Simulink
**求解器设置**：离散（Discrete），步长0.0001s（100μs）
**仿真时间**：20s
**停止时间**：20s

### 顶层结构

模型顶层包含两个核心子系统，通过直流线路互联：

```
┌──────────────────┐     直流正极线     ┌──────────────────┐
│                  │ ═══════════════════ │                  │
│  MMC1定直流电压    │                    │  MMC2定有功功率1   │
│  （A站/主控站）    │                    │  （B站/从控站）    │
│                  │ ═══════════════════ │                  │
└──────────────────┘     直流负极线     └──────────────────┘
```

- **MMC1定直流电压**：主控站，负责维持直流母线电压恒定
- **MMC2定有功功率1**：从控站，控制有功功率的传输

## MMC1定直流电压子系统详解

### 内部模块构成

从截图分析，MMC1内部包含以下关键模块：

**1. 交流侧电源与变压器**
- 三相交流电源（左下角）
- 升压/降压变压器（接线组别区分）
- 三相VI测量模块（Three-Phase V-I Measurement）

**2. MMC本体（中央灰色大块）**
- 多端口子系统，包含6个桥臂（三相上下桥臂）
- 输入信号：6组PWM开关信号（PWMAP, PWMAL, PWMBP, PWMBL, PWMCP, PWMCL）
- 输出信号：
  - 三相交流电压电流（IAP, IAL, IBP, IBL, ICP, ICL）
  - 三相阀侧电压（VAP, VBP, VCP）
  - 子模块电容电压（VCU, VBU, VAL, VDC等）
  - 直流侧电压电流（IDC, VDC, VDCS）
- 100:10变比（可能是100kV:10kV或对应标幺值）

**3. NLM调制模块**
- 最近电平逼近调制（NLM）子系统
- 输入：参考电压波形（VA, VB, VC来自控制环路）+ 子模块电容电压
- 输入信号包括：[VAL], [VBL], [VCL], [HLA], [HLB], [HLC] 等
- 输出：6组PWM脉冲信号给MMC本体
- 内含排序均压算法

**4. 控制系统（右下区域）**
- **PLL锁相环**：从交流侧电压获取同步角度
- **外环控制**：
  - 直流电压环（定Udc）：子模块电容电压→直流电压调节
  - 目标：[VAP] 端口输出稳定直流电压
- **内环电流控制**：
  - d轴电流控制（有功）
  - q轴电流控制（无功）
  - 输出参考电压给NLM
- **坐标变换**：abc/dq变换模块

**5. 测量与显示（右侧）**
- Scope观测器（V1, V2端子）
- 直流侧电压电流测量
- 交流侧P/Q功率计算（Vabc→P, Iabc→Q → Scope19）
- 信号输出到工作空间

### 关键信号标注

| 信号名 | 含义 | 位置 |
|--------|------|------|
| [AP], [AL] | A相上/下桥臂电流 | MMC输出→右侧 |
| [BP], [BL] | B相上/下桥臂电流 | MMC输出→右侧 |
| [CP], [CL] | C相上/下桥臂电流 | MMC输出→右侧 |
| [VAP], [VBP], [VCP] | 阀侧三相电压 | MMC输出 |
| [VCU], [VBU] | 子模块电容电压 | MMC输出→NLM |
| [VAL], [VDC] | 交流阀侧电压/直流电压 | 测量→控制 |
| [IDC] | 直流电流 | 测量输出 |
| PWMAP~PWMCL | 6组PWM脉冲 | NLM→MMC |
| HLA, HLB, HLC | 桥臂参考电压 | 控制→NLM |
| dref, qref | dq轴电流参考值 | 外环→内环 |

## MMC2定有功功率子系统详解

### 结构对比

MMC2的内部结构与MMC1基本一致，包含相同的模块：
- MMC本体（三相六桥臂）
- NLM调制模块
- 三相交流电源 + 变压器
- 控制系统

### 关键差异

**外环控制不同**：
- MMC1外环：**定直流电压**（Udc_ref → PI → id_ref）
- MMC2外环：**定有功功率**（P_ref → PI → id_ref）

MMC2中可以看到：
- 功率参考值信号（Mds0, Mqs0 等）用于设定有功功率目标
- 外环接收有功功率测量值（wt），通过PI调节器产生d轴电流参考
- 区别于MMC1的直流电压反馈环路

### 测量模块

与MMC1类似：
- Vabc2, Iabc2 → P/Q功率计算 → Scope19
- 直流侧电压、电流测量输出

## 仿真结果解读

### MMC1侧P/Q波形特征（MMC1定直流电压站）

从仿真结果截图分析（图：P,Q波形，y轴×10⁹，x轴0-20s）：

**有功功率wt（黑色曲线）：**
- 稳态值约 0.3×10⁹ W = 300MW
- 暂态过程：0-2s内有较大振荡（初始过冲至4×10⁹）
- 2s后逐渐收敛，带有持续的纹波

**无功功率Q（橙色曲线）：**
- 稳态值约 1.0×10⁹ Var = 1000MVar（偏大）
- 暂态过程中有负向过冲至-1.5×10⁹
- 纹波较大，包络宽约0.3×10⁹

**诊断要点**：
- 暂态时间约2-4s，对于MMC系统属于偏长
- 无功功率稳态值偏大（1GVar），需检查无功参考值设定
- 持续纹波可能与子模块数不足或电容值偏小有关

### MMC2侧P/Q波形特征（MMC2定有功功率站）

从仿真结果截图分析（图：P,Q波形，y轴×10⁸，x轴0-20s）：

**有功功率wt（黑色曲线）：**
- 稳态值约 -0.5×10⁸ W = -50MW（负值表示从直流侧吸收功率传到交流侧）
- 暂态过程：0-4s有明显振荡，最大负向过冲至-8×10⁸
- 4s后基本稳定

**无功功率Q（橙色曲线）：**
- 稳态值接近0（符合定Q=0的设定）
- 暂态过程中有正向过冲

**诊断要点**：
- MMC2的功率量级（~50MW）与MMC1（~300MW）差异较大，需确认设计容量
- 暂态振荡收敛时间约4s，可通过调节PI参数改善
- 无功基本为零，说明Q环控制正常

## 如何在MATLAB中查看详细参数

1. 打开MMCSIDUAN0x11.slx模型
2. 双击"MMC1定直流电压"子系统进入
3. 双击MMC本体（灰色大块）→查看子模块数、电容值
4. 查看桥臂电感：双击电感模块读取L值
5. 查看控制器参数：双击PI模块读取Kp、Ki
6. 查看NLM模块：双击进入查看调制算法参数
7. 查看交流源参数：双击三相源查看电压幅值和频率

## MMCSIDUAN2 模型版本说明

### 模型概述

`MMCSIDUAN2.slx` 是 `MMCSIDUAN0x11.slx` 的改进版本，两者共享相同的顶层架构：

- **求解器**：Discrete，步长 0.0001s（100μs）
- **顶层结构**：MMC1定直流电压 + MMC2定有功功率1，通过直流正/负极线连接
- **仿真时间**：停止时间 20s

### 与 MMCSIDUAN0x11 的关键差异

| 对比项 | MMCSIDUAN0x11 | MMCSIDUAN2 |
|--------|---------------|------------|
| 求解器 | Discrete 0.0001s | Discrete 0.0001s |
| 顶层结构 | 相同 | 相同 |
| 环流抑制器 | 有 | 有（环流抑制器1） |
| 坐标变换模块 | 有 | 有 |
| 文件来源 | 基础版本 | 改进/调试版本 |

## 补充模块详解（基于高清截图）

以下内容基于 MMCSIDUAN0x11 / MMCSIDUAN2 模型的高清截图补充，涵盖原文档未详细描述的模块。

### 环流抑制器1（Circulating Current Suppression Controller, CCSC）

**位置**：MMC1 和 MMC2 子系统内部，控制区域左下方

**输入信号**：
- [wt]：锁相环输出角度
- [IDC]：直流电流
- [IAP], [IAL]：A相上/下桥臂电流
- [IBP], [IBL]：B相上/下桥臂电流
- [ICP], [ICL]：C相上/下桥臂电流

**输出信号**：
- HLA → [HLA]：A相桥臂环流抑制参考电压
- HLB → [HLB]：B相桥臂环流抑制参考电压
- HLC → [HLC]：C相桥臂环流抑制参考电压

**功能说明**：
- 桥臂环流是MMC固有问题，以2倍基频为主要分量
- CCSC从桥臂电流中提取环流分量，通过PI调节器产生补偿电压
- 补偿电压叠加到NLM调制的参考电压上，抑制环流

### 坐标变换模块

**位置**：MMC1 和 MMC2 子系统内部，控制区域左下方

**输入信号**：
- Vabc / Vabc1 / Vabc2：三相交流电压采样
- Iabc / Iabc1 / Iabc2：三相交流电流采样

**输出信号**：
- [id], [iq]：dq轴电流分量
- [ed], [eq]：dq轴电压分量
- [wt]：锁相环角度

**功能说明**：
- 将ABC三相静止坐标系变换到dq旋转坐标系
- 内含PLL锁相环，从交流侧电压提取同步角度wt
- d轴控制有功功率，q轴控制无功功率

### 直流电压测量回路

**位置**：MMC本体右侧

**结构**：
- VDC+ 和 VDC- 端口分别接正/负直流母线
- 通过加法器（端口1: +, 端口2: -）获取正负极电压差
- 变比 100:10（电压互感器模型）
- 输出信号 [Vdc]

**显示值**：3.405e+04（即 34,050V ≈ 34.05kV），结合 100:10 变比，实际直流电压约 340.5kV（±170.25kV）

### VDC 参考值

**位置**：控制系统底部

**设定值**：20000（常数模块）
- 该值为直流电压参考值，用于外环控制
- 与测量变比关联：实际母线电压 = 20000 × 变比系数
- 对于MMC1，20000 作为定直流电压环的 Vdc_ref

### NLM 调制模块详细信号

**输入端口**：
| 端口 | 信号 | 含义 |
|------|------|------|
| VA | [VA] | A相参考电压 |
| VB | [VB] | B相参考电压 |
| VC | [VC] | C相参考电压 |
| HLA | [HLA] | A相环流抑制补偿 |
| HLB | [HLB] | B相环流抑制补偿 |
| HLC | [HLC] | C相环流抑制补偿 |

**输出端口**：
| 端口 | 信号 | 含义 |
|------|------|------|
| VAP | [vap] | A相上桥臂参考（含补偿） |
| VAL | [val] | A相下桥臂参考 |
| VBP | [vbp] | B相上桥臂参考 |
| VBL | [vbl] | B相下桥臂参考 |
| VCP | [vcp] | C相上桥臂参考 |
| VCL | [vcl] | C相下桥臂参考 |

**内部逻辑**（fcn函数块）：
- 将参考电压 VA/VB/VC 与环流补偿 HLA/HLB/HLC 叠加
- 执行最近电平逼近（NLM）调制算法
- 结合子模块排序均压策略输出开关信号

### 测量与显示模块（Scopes）

从截图可确认以下测量点：

| Scope 名称 | 接入信号 | 用途 |
|------------|----------|------|
| Vinv / Vinv2 | 阀侧三相电压 | 观察MMC输出交流电压波形 |
| Iinv / Iinv2 | 阀侧三相电流 | 观察MMC输出电流波形 |
| A相上桥臂电流 | [IAP] | 观察单桥臂电流，判断环流大小 |
| 子模块电容电压 | [VAP] | 观察子模块电容电压波动 |
| 直流电压瞬聊 | [wt] / Vdc | 观察直流电压动态 |
| 交流电流瞬聊 | 交流侧电流 | 观察交流电流动态 |
| V1, V2 | 直流正负极端口 | 直接测量点 |

### P/Q 功率计算模块

**位置**：MMC2 控制区右下方

**结构**：
- 输入：Vabc2（三相电压）、Iabc2（三相电流）
- 内部：abc → P 和 abc → Q 计算
- 输出：P（有功功率）、Q（无功功率）→ Scope
- **Q参考值 = 0**（由常数模块设定），表示MMC2不输出/吸收无功

### 变压器参数（Three-Phase V-I Measurement2）

- 接线组别：Yg/D11（Y接地/三角形，11点钟方向）
- 位于交流源和MMC本体之间
- 带三相电压电流测量功能（V-I Measurement）
- 变比与系统电压等级对应

### 内环控制信号汇总

在底部控制区域可见的信号连接：

```
[VDC] ──→ 外环PI ──→ id_ref
                                    ├──→ 内环d轴PI ──→ Vd_ref ──→ [VA]
[id] ──────────────────────→ ─┘
[iq] ──────────────────────→ ─┐
                                    ├──→ 内环q轴PI ──→ Vq_ref ──→ [VB]
Q_ref(0) ──→ 外环PI ──→ iq_ref ┘
[ed], [eq] ──→ 前馈解耦 ──→ 叠加到Vd/Vq参考
[ed1] ──→ 附加解耦项
Vabc, Iabc ──→ 坐标变换 ──→ [id],[iq],[ed],[eq],[wt]
dref ──→ 输出参考信号
```

**MMC1（定直流电压）外环**：VDC测量 → 与Vdc_ref(20000)比较 → PI → id_ref
**MMC2（定有功功率）外环**：P测量 → 与P_ref比较 → PI → id_ref；Q_ref = 0 → PI → iq_ref
