# MMC与柔性直流基础理论

## 柔性直流输电概述

柔性直流输电（VSC-HVDC）是基于电压源换流器（VSC）的高压直流输电技术。与传统直流输电（LCC-HVDC）相比，核心优势：

- **无换相失败**：不依赖交流电网换相，可向无源网络供电
- **有功无功解耦控制**：独立调节P和Q，互不影响
- **黑启动能力**：可在无交流电源情况下启动系统
- **谐波含量低**：MMC拓扑下THD < 0.3%，无需交流滤波器
- **占地面积小**：无需大量滤波设备和无功补偿装置

## 模块化多电平换流器（MMC）

### MMC拓扑结构

MMC是柔直系统的核心设备。每相由上桥臂和下桥臂组成，每个桥臂包含N个子模块和一个桥臂电感串联。

**三相六桥臂结构：**
```
           Udc/2
            │
    ┌───────┼───────┐
    │       │       │
  [SM]×N  [SM]×N  [SM]×N    ← 上桥臂（含L_arm）
  [L_arm] [L_arm] [L_arm]
    │       │       │
    ├──A    ├──B    ├──C     ← 交流输出端
    │       │       │
  [L_arm] [L_arm] [L_arm]
  [SM]×N  [SM]×N  [SM]×N    ← 下桥臂（含L_arm）
    │       │       │
    └───────┼───────┘
            │
          -Udc/2
```

### 子模块类型

**半桥子模块（HB-SM）**：
- 2个IGBT + 2个反并联二极管 + 1个直流电容
- 三种工作状态：
  - 投入（Insert）：T1开T2关 → 输出电容电压Uc
  - 切除（Bypass）：T1关T2开 → 输出0V
  - 闭锁（Block）：T1关T2关 → 电容浮动
- 优点：损耗低、成本低，应用最广
- 缺点：不具备直流故障自清除能力

**全桥子模块（FB-SM）**：
- 4个IGBT + 4个反并联二极管 + 1个直流电容
- 可输出+Uc、0、-Uc三种电平
- 优点：具备直流故障自清除能力
- 缺点：损耗较高、成本约为半桥的1.5倍

**混合子模块**：半桥与全桥组合（如甘孜工程36全桥+36半桥），兼顾经济性与故障清除能力。

### 子模块电容电压

子模块电容额定电压：

$$U_{c0} = \frac{U_{dc}}{N}$$

其中 $U_{dc}$ 为直流母线电压，$N$ 为每桥臂子模块数。

电容储能要求：

$$E_{cap} = \frac{1}{2} C_{sm} U_{c0}^2$$

## VSC有功无功控制原理

### 功率控制基础

VSC通过控制输出电压的幅值和相位，实现有功和无功的独立调节：

$$P = \frac{U_{vsc} \cdot U_{grid}}{X} \sin(\delta)$$

$$Q = \frac{U_{vsc}^2 - U_{vsc} \cdot U_{grid} \cos(\delta)}{X}$$

- $U_{vsc}$：换流器输出电压幅值（可控）
- $U_{grid}$：电网电压幅值
- $\delta$：换流器电压超前电网电压的相位差
- $X$：等效电抗（主要为桥臂电感和变压器漏抗）

**控制规律：**
- 调节 $\delta$ → 控制有功功率P
- 调节 $U_{vsc}$ → 控制无功功率Q
- P和Q完全解耦

### dq坐标系矢量控制

实际控制采用Park变换将三相交流量转换到dq旋转坐标系：

$$\begin{cases} P = \frac{3}{2}(u_d i_d + u_q i_q) \\ Q = \frac{3}{2}(u_q i_d - u_d i_q) \end{cases}$$

当PLL锁定电网电压方向后，$u_q = 0$，简化为：

$$\begin{cases} P = \frac{3}{2} u_d i_d \\ Q = -\frac{3}{2} u_d i_q \end{cases}$$

即：**id控制有功功率，iq控制无功功率**。

## 两端柔直系统控制策略

### A站（主控站）— 定直流电压控制

A站负责维持直流母线电压稳定，是整个系统的功率平衡节点。

**控制环路：**
```
Udc_ref ──→ [PI外环] ──→ id_ref ──→ [PI内环] ──→ 调制信号
                │                        │
            Udc_meas ←─ 测量           id_meas ←─ 测量
```

**物理意义**：直流母线电压反映功率平衡状态：
$$P_{in} - P_{out} = C \frac{dU_{dc}}{dt} U_{dc}$$

当输入功率 > 输出功率时，$U_{dc}$ 升高，A站外环PI输出增大id_ref，增加有功吸收；反之亦然。

### B站（从控站）— 定有功功率控制

B站按照调度指令控制输出/吸收的有功功率。

**控制环路：**
```
P_ref ──→ [PI外环] ──→ id_ref ──→ [PI内环] ──→ 调制信号
              │                        │
          P_meas ←─ 计算             id_meas ←─ 测量
```

### 无功控制（两站均有）

两站均可独立控制无功功率：
- 定Q模式：$Q_{ref}$ → PI控制器 → $i_{q\_ref}$
- 定V模式：$U_{ac\_ref}$ → PI控制器 → $i_{q\_ref}$（用于暂态电压支撑）

## NLM调制与子模块均压

### 最近电平逼近调制（NLM）

NLM是MMC最常用的调制方式，适用于子模块数较多的场合。

**原理**：将期望的桥臂电压除以子模块电容电压，四舍五入得到需要投入的子模块数：

$$n_{insert} = \text{round}\left(\frac{u_{arm\_ref}}{U_{c0}}\right)$$

**优点**：
- 开关频率低（基频开关），损耗小
- 实现简单，无需载波
- 输出电压阶梯波接近正弦

### 子模块均压排序算法

每个开关周期，对同一桥臂的所有子模块电容电压排序：
1. 若桥臂电流为正（电容充电方向），投入电压**最低**的子模块
2. 若桥臂电流为负（电容放电方向），投入电压**最高**的子模块

### 桥臂环流抑制

MMC桥臂间存在环流（主要为2倍基频分量），增加损耗并影响子模块电压平衡。

**抑制方法**：在内环控制中增加环流抑制控制器（CCSC），提取2ω分量并生成补偿信号。

## 直流故障特性与保护

### 极间短路故障

故障发生后：
1. 子模块电容通过二极管快速放电（μs级）
2. 故障电流上升率极高（>50A/μs）
3. 交流侧通过二极管续流，持续馈入故障电流

### 保护方案

| 方案 | 响应时间 | 成本 | 适用 |
|------|---------|------|------|
| 半桥MMC + 交流断路器 | 20-60ms | 低 | 对速度要求不高 |
| 半桥MMC + 直流断路器 | 2-5ms | 高 | 多端直流电网 |
| 全桥MMC自清除 | <1ms | 最高 | 要求极高可靠性 |
| 故障限流器（FCL） | 灵活 | 中 | 辅助限流 |

## PLL锁相环

PLL用于锁定交流电网电压的频率和相位，提供dq变换的同步角θ。

**SRF-PLL（同步旋转坐标系PLL）：**
```
Vabc → Park变换(θ) → uq → PI → ω → 积分 → θ
                                        ↑
                                     反馈到Park变换
```

当PLL锁定时，$u_q = 0$，此时d轴与电网电压矢量对齐。

**注意事项**：
- 弱电网下PLL带宽不宜过高（容易与电网阻抗产生谐振）
- 典型带宽：20-50Hz
