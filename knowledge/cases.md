# 工程案例 — 甘孜光伏与四川柔直配电

## 甘孜大规模全光伏发电基地

### 项目概况

- **位置**：甘孜州乡城、巴塘、得荣三县交界处（横断山脉腹地）
- **海拔**：3500-4500m
- **太阳能资源**：年均太阳总辐射量5900-6100 MJ/m²，属全国高太阳能地区
- **规划装机容量**：
  - 近期（~2025年）：966-1450 MW
  - 中期（2025-2030年）：1655-2520 MW
  - 远期（2030年后）：3981-5632 MW

### 推荐直流组网方案

**方案3（推荐）：直流汇集-直流分散送出**

```
光伏阵列(DC 1500V)
      ↓ DC/DC (2MW, IPOS拓扑)
光伏簇群汇集(±20kV)
      ↓ 直流升压站(250MW)
直流母线(±150kV)
      ↓ 8回±150kV直流线路
MMC逆变站(2台并联, 850MW)
      ↓
AC 500kV → 甘南汇集站 → 主网
```

| 级别 | 电压 | 功能 | 设备 |
|------|------|------|------|

### 储能配置

- **配置原则**：4h以上容量型储能，兼具平滑波动
- **储能类型**：磷酸铁锂电池
- **配置比例**：20%-30%装机容量
- **循环寿命**：8000次
- **充放电效率**：94%
- **典型配置**：50MW/100MWh（单站）

## 四川柔性直流配电应用场景

### 场景一：农村配网

**问题**：
- 分布式光伏大量接入 → 电压越限50%以上用户
- 供电半径长 → 电压损失大
- 设备老旧 → 供电可靠性低
- 三相不平衡率 > 95%

**柔直方案**：
- 提高供电电压增加半径减少损耗
- 减少光伏DC/AC换流环节
- 台区能量互通互济
- 源网荷储协调

**关键指标**：
- 供电可靠率：99.845%
- 综合电压合格率：99.804%
- 户均配变容量：2.32 kVA

### 场景二：直流源荷高渗透率园区

**标杆项目**：成都天府国际生物城
- 占地44km²，2024年6月全面投运
- ±10kV双环网，地埋率92%
- 最大负荷312MW，光伏最大出力288MW
- 午间就地消纳率93%，弃光率0.9%
- 直流供电路径效率91%（比交流高8%）

**核心参数**：
| 参数 | 数值 |
|------|------|
| 直流电压 | ±10kV |
| 电缆 | 2×400mm²铜芯XLPE |
| 最大输送电流 | 2.2 kA |
| 极限容量 | 48 MW |
| N-1后容量 | 36 MW |
| 低压母线 | 750V / 375V两级 |

**经济性**：
- 单位投资1.25元/W（较2020年下降38%）
- 储能综合收益0.78元/kWh
- 静态回收期5.4年

### 场景三：集中式新能源送出

**四川送出通道**：
- 已建成±800kV特直通道3回（雅中-江西、白鹤滩-江苏/浙江）
- 每回容量8GW，总长5100km
- 2024年批复"甘眉-成渝"第四回±800kV通道

**甘孜800kV柔直送出站**：
- 全球首套±800kV/5GW全对称单极柔直成套装备
- 6.5kV/3kA IGCT-PWM器件
- 150ms内直流故障清除并重启
- 80ms可切换至STATCOM模式，输出2.5Gvar动态无功

### 场景四：交直流柔性互联

**典型工程**：

| 工程 | 技术特点 | 效果 |
|------|---------|------|
| 深圳柔直配网 | 110kV交流→±10kV直流 | 双电源手拉手 |
| 杭州大江东 | 20kV/10kV不同供区互联 | 不增加短路容量 |
| 南京直流供电 | 四端口环状直流供电 | 高可靠性 |
| 温州平阳园区 | ±10kV混合直流(LCC+VSC) | 直流直供电镀 |
| 川渝背靠背 | 分区异步互联 | 故障隔离 |

**柔性互联装置控制方式**：

| 控制方式 | 特点 | 优缺点 |
|---------|------|--------|
| 主从控制 | 一主站定Udc，其余定P | 简单但依赖通信 |
| 电压裕度控制 | 多级功率平衡点 | 降低通信需求但切换震荡 |
| 下垂控制 | 各站自我调节 | 无需通信但Udc有偏差 |
| 分段下垂 | 正常定Udc+故障时下垂 | 综合最优 |
